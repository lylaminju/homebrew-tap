# homebrew-tap

Custom Homebrew tap by [lylaminju](https://github.com/lylaminju).

## Usage

```bash
brew tap lylaminju/tap
```

## Available Casks

| Cask | Description | Install |
|------|-------------|---------|
| [claude-token-cat](https://github.com/lylaminju/claude-token-cat) | macOS menu bar app for Claude usage tracking | `brew install --cask lylaminju/tap/claude-token-cat` |
